# Helpi Brand Quick Reference

## Brand Core
- Personality: compassionate, reliable, energetic, transparent.
- Message style: emotion first, proof second, action third.
- Audience feel: "I trust this organization and I know what to do next."

## Color Quick Set
- Primary text: #202E3F
- Strong text: #0F0F0F
- Body text: #4D5561
- Muted text: #6F6F6F
- Base background: #FFFFFF
- Warm background: #FFF9F8
- Warm accent surface: #FFF1EE
- Divider/border: #C5C6C6 / #F0F0F0

Usage:
- Headlines on light surfaces use #202E3F.
- Long body copy defaults to #4D5561.
- Warm sections (#FFF9F8 / #FFF1EE) frame donation and mission blocks.

## Typography Quick Set
- Display and headings: Questrial
- Body: Questrial
- UI/meta text: Inter
- Editorial accent: Petrona italic

Recommended sizes:
- H1: 50 desktop, 36 mobile
- H2: 36 desktop, 28 mobile
- Body: 16 to 18
- Label/meta: 14

Line-height:
- Headings: 1.1 to 1.2
- Body: 1.5

## Layout and Spacing
- Max container: 1345px
- Breakpoints: 768, 992, 1280
- Core spacing rhythm: 16, 24, 30, 40, 70, 80, 130
- Default card radius: 20

## Component Style Rules
- CTAs: short imperative labels (Donate, Join, Contact, View More).
- Cards: image/icon anchor, clear title, concise context, obvious action.
- Metrics: large value + short descriptor (e.g., 200K+ Lives Transformed).
- FAQ rows: clear question with plus affordance and strong active state.

## Motion
- Default timing: 300ms
- Easing: cubic-bezier(.44, 0, .56, 1)
- Motion should guide focus, not decorate.

## Imagery and Iconography
- Use authentic community/volunteer photography.
- Prioritize emotional clarity over abstract stock visuals.
- Icons should support labels, not replace them.

## Copy Voice
- Warm, direct, hopeful.
- Plain language over nonprofit jargon.
- CTA copy should be explicit and short.

## Accessibility Snapshot
- Maintain WCAG AA contrast for text and controls.
- Keep touch/click targets at least 44x44.
- Preserve logical heading order and keyboard focus visibility.
- Provide alt text for all meaningful imagery.
